# StyleCast ERP Backend

Multi-tenant ERP backend for fashion brands built with FastAPI, PostgreSQL, and SQLAlchemy.

## Features

- Brand management
- User authentication with JWT
- Product catalog management
- Product variants
- Inventory tracking
- Order management
- Shipping rules
- Analytics APIs

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic
- JWT Authentication

## API Modules

- /brands
- /auth
- /products
- /variants
- /inventory
- /orders
- /shipping-rules
- /analytics
